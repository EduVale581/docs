# Clientes

Gestiona toda la información y relaciones comerciales con tus clientes.

## ¿Qué es un cliente?

Un cliente es una persona natural o jurídica que adquiere bienes o servicios de tu empresa.

## Información del cliente

### Datos básicos

- **RUT**: Rol Único Tributario
- **Razón social**: Nombre legal del cliente
- **Nombre de fantasía**: Nombre comercial
- **Giro**: Actividad económica principal

### Datos de contacto

- **Dirección de facturación**: Para envío de documentos
- **Dirección de despacho**: Para entrega de productos
- **Teléfonos**: Números de contacto
- **Email**: Correo electrónico
- **Contacto principal**: Persona de contacto

### Información comercial

- **Condiciones de pago**: Plazo y forma de pago
- **Descuentos**: Descuentos especiales acordados
- **Límite de crédito**: Monto máximo autorizado
- **Lista de precios**: Precios especiales si corresponde

## Tipos de clientes

- **Cliente final**: Consumidor final del producto/servicio
- **Cliente corporativo**: Empresas que compran para reventa
- **Cliente mayorista**: Compra grandes volúmenes
- **Cliente minorista**: Compra para venta al detalle

## Estado del cliente

- **Activo**: Cliente habilitado para ventas
- **Inactivo**: Temporalmente suspendido
- **Moroso**: Con pagos pendientes
- **Bloqueado**: No disponible para nuevas ventas

## Segmentación de clientes

### Por volumen de compras

- **VIP**: Clientes de alto volumen
- **Premium**: Clientes importantes
- **Estándar**: Clientes regulares
- **Ocasional**: Clientes esporádicos

### Por ubicación geográfica

- **Local**: Misma ciudad
- **Regional**: Misma región
- **Nacional**: Otras regiones del país
- **Internacional**: Clientes extranjeros

## Análisis de clientes

### Indicadores clave

- **Volumen de ventas**: Total de compras
- **Frecuencia**: Periodicidad de compras
- **Margen**: Rentabilidad por cliente
- **Antigüedad**: Tiempo como cliente
- **Pagos**: Comportamiento de pago
