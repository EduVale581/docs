# Libro Mayor

El libro mayor muestra todos los movimientos contables agrupados por cuenta contable.

## ¿Qué es el libro mayor?

El libro mayor presenta los movimientos de cada cuenta contable por separado, mostrando débitos, créditos y saldos.

## Funcionalidades

- **Filtrar por cuenta**: Ve solo las cuentas que necesites
- **Rango de fechas**: Selecciona el período a consultar
- **Saldos acumulados**: Ve la evolución de cada cuenta
- **Exportación**: Descarga en diferentes formatos
- **Drill-down**: Ve el detalle de cada movimiento

## Estructura del mayor

Para cada cuenta muestra:

1. **Código y nombre de cuenta**
2. **Saldo inicial**
3. **Movimientos del período**:
   - Fecha
   - Número de asiento
   - Descripción
   - Debe
   - Haber
   - Saldo acumulado
4. **Totales del período**
5. **Saldo final**
